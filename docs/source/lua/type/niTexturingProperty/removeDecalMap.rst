removeDecalMap
====================================================================================================

Attempts to remove a decal at a given index.

Returns
----------------------------------------------------------------------------------------------------

`boolean`_. True if the decal was removed.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

index (`number`_)
    The index of the decal to remove.

.. _`number`: ../../../lua/type/number.html
