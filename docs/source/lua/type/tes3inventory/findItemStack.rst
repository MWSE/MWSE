findItemStack
====================================================================================================

Searches for an item stack in the inventory.

Returns
----------------------------------------------------------------------------------------------------

`tes3itemStack`_.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

item (`tes3item`_)
    The item to search for.
