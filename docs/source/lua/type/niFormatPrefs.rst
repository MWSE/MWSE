niFormatPrefs
====================================================================================================

A class that represents a set of preferences for texture formats. These preferences dictate levels of pixel accuracy.

Properties
----------------------------------------------------------------------------------------------------

`alphaLayout`_ (`niTextureFormatPrefsAlphaFormat`_)
    The Alpha format.

`mipLayout`_ (`niTextureFormatPrefsMipFlag`_)
    The MIP format.

`pixelLayout`_ (`niTextureFormatPrefsPixelLayout`_)
    The pixel layout.
