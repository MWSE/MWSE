tes3packedColor
====================================================================================================

A densely packed structure for single-byte red, green, blue and alpha values.

Properties
----------------------------------------------------------------------------------------------------

`a`_ (`number`_)
    The alpha color value.

`b`_ (`number`_)
    The blue color value.

`g`_ (`number`_)
    The green color value.

`r`_ (`number`_)
    The red color value.
