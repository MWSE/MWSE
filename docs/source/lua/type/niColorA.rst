niColorA
====================================================================================================

An object that represents a RGB color with alpha channel support.

----------------------------------------------------------------------------------------------------

`g`_ (`number`_)

`green`_ (`number`_)
    Alias for the green value of the color.

`r`_ (`number`_)
    The red value of the color.
