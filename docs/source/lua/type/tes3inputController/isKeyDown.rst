isKeyDown
====================================================================================================

Performs a key down test for a given scan key code.

Returns
----------------------------------------------------------------------------------------------------

`boolean`_.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

key (`number`_)
    The scan code to test. Constants available through ``tes3.scanCode``.

.. _`number`: ../../../lua/type/number.html
