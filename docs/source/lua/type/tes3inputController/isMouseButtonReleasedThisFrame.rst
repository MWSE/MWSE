isMouseButtonReleasedThisFrame
====================================================================================================

Returns true if a mouse button was down last frame, and up this frame.

Returns
----------------------------------------------------------------------------------------------------

`boolean`_.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

button (`number`_)
    The mouse button to test. Must be between 0 and 7.
