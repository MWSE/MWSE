tes3cellExteriorData
====================================================================================================

A structure that contains a cell and its grid coordinates.

Properties
----------------------------------------------------------------------------------------------------

`cell`_ (`tes3cell`_)
    The cell in the package.

`gridX`_ (`number`_)
    The cell in the package's X grid value.

`gridY`_ (`number`_)
    The cell in the package's Y grid value.
