tes3lockNode
====================================================================================================

An attachment-capable structure that maintains lock and trap data.

Properties
----------------------------------------------------------------------------------------------------

`key`_ (`tes3misc`_)
    The key that will open this door.

`level`_ (`number`_)
    The level of the lock.

`locked`_ (`boolean`_)
    The locked state.

`trap`_ (`tes3spell`_)
    The trap associated with the object.
