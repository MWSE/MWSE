unequip
====================================================================================================

Unequips one or more items from the actor.

Returns
----------------------------------------------------------------------------------------------------

`boolean`_.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters through a table with the given keys:

item (`tes3item`_, `string`_)
    Optional. The item to unequip.

type (`number`_)
    Optional. The item type to unequip. Only used if no other parameter is provided.

armorSlot (`number`_)
    Optional. The armor slot to unequip.

clothingSlot (`number`_)
    Optional. The clothing slot to unequip.
