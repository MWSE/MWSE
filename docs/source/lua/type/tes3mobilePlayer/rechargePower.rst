rechargePower
====================================================================================================

Makes a power immediately available for casting again.

Returns
----------------------------------------------------------------------------------------------------

`number`_.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

power (`tes3spell`_)
    The spell object for the power.
