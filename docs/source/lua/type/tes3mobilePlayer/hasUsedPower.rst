hasUsedPower
====================================================================================================

Check if a power has been used and is recharging.

Returns
----------------------------------------------------------------------------------------------------

`boolean`_.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

power (`tes3spell`_)
    The spell object for the power.
