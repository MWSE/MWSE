calculateCastChance
====================================================================================================

Calculates the chance that a caster can cast a given spell.

Returns
----------------------------------------------------------------------------------------------------

`number`_.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters through a table with the given keys:

checkMagicka (`boolean`_)
    Optional. Determines if the caster's magicka should be taken into account during the calculation.

caster (`tes3reference`_, `tes3mobileActor`_)
    The caster to perform the calculation against.
