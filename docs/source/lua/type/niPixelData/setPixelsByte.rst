setPixelsByte
====================================================================================================

Sets the pixel data from byte data.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

data (`table`_)
    The byte data to be set to.

mipMapLevel (`number`_)
    Optional. Default value: 1
