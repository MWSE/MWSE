contains
====================================================================================================

Checks if the specified spell is contained in the spell list.

Returns
----------------------------------------------------------------------------------------------------

`boolean`_.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

spell (`string`_, `tes3spell`_)
    The spell.
