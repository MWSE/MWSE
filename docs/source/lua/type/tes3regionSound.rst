tes3regionSound
====================================================================================================

A structure that contains region sound information.

Properties
----------------------------------------------------------------------------------------------------

`chance`_ (`number`_)
    The chance the region sound will be used. Must be an integer between 0 - 100.

`sound`_ (`tes3regionSound`_)
    The region sound's sound.
