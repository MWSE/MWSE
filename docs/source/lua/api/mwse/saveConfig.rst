mwse.saveConfig
====================================================================================================

Saves a config table to Data Files\MWSE\config\{fileName}.json.

----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

fileName (`string`_)
    No description available.

object (`unknown`_)
