mge.freeHUD
====================================================================================================

Frees a HUD element. If a HUD ID is provided, this is a warpper for MGEFreeHUD, otherwise it is a wrapper for MGENIDFreeHUD.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

hud (`string`_)
    Optional. No description available.
