table.removevalue
====================================================================================================

Removes a value from a given table. Returns true if the value was successfully removed.

Returns
----------------------------------------------------------------------------------------------------

`boolean`_.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

t (`table`_)
    No description available.
