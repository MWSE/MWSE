table.size
====================================================================================================

Returns the number of elements inside the table. Unlike the length operator (#) this will work with any table.

Returns
----------------------------------------------------------------------------------------------------

`number`_.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

t (`table`_)
