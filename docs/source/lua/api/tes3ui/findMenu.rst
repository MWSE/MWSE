tes3ui.findMenu
====================================================================================================

Locates a top-level menu through its id.

Returns
----------------------------------------------------------------------------------------------------

`tes3uiElement`_.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

id (`number`_)
    The ID of the menu to locate.
