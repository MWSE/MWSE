lfs.directoryexists
====================================================================================================

Simple check to determine if a directory exists.

Returns
----------------------------------------------------------------------------------------------------

`boolean`_. true if the directory exists.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

path (`string`_)
    The path to determine existance of.
