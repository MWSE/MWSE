lfs.fileexists
====================================================================================================

Simple check to determine if a file exists.

Returns
----------------------------------------------------------------------------------------------------

`boolean`_. true if the file exists.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

path (`string`_)
    The path to determine existance of.
