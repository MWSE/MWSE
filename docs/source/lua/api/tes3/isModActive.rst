tes3.isModActive
====================================================================================================

Determines if the player has a given ESP or ESM file active.

Returns
----------------------------------------------------------------------------------------------------

`boolean`_.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

filename (`string`_)
    The filename of the mod to find, including the extension.
