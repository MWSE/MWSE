tes3.saveGame
====================================================================================================

Saves the game.

Returns
----------------------------------------------------------------------------------------------------

`boolean`_.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters through a table with the given keys:

file (`string`_)
    Default: ``"quiksave"``. The filename of the save that will be created, without extension.

name (`string`_)
    Default: ``"Quicksave"``. The display name of the save.
