tes3.getItemIsStolen
====================================================================================================

This function checks item's stolen flag.

Returns
----------------------------------------------------------------------------------------------------

`boolean`_.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

item (`tes3item`_)
    The item to check.

from (`tes3baseObject`_)
    Where the item was stolen from.
