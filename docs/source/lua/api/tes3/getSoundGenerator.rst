tes3.getSoundGenerator
====================================================================================================

Returns a sound generator by a given creature id and type.

Returns
----------------------------------------------------------------------------------------------------

`tes3soundGenerator`_.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

creatureId (`string`_)
    No description available.

soundType (`number`_)
    Maps to tes3.soundGenType constants.
