determinedAction
====================================================================================================

This event fires when an actor has determined an action in a combat session.

Related events: `determineAction`_

Event Data
----------------------------------------------------------------------------------------------------

session
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

`tes3combatSession`_. Read-only. The combat session the action has been determined for.
