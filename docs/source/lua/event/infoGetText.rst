infoGetText
====================================================================================================

This event fires when a dialogue info object text is retrieved.

Event Data
----------------------------------------------------------------------------------------------------

info
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

`tes3dialogueInfo`_. Read-only. The dialogue info object.

loadOriginalText
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

method. Loads the original text for the info from the game files, bypassing this event.
