cellDeactivated
====================================================================================================

This event is triggered when a cell is deactivated.

Related events: `cellActivated`_, `cellChanged`_

Event Data
----------------------------------------------------------------------------------------------------

cell
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

`tes3cell`_. Read-only. The cell which was deactivated.
