collideWater
====================================================================================================

This event is triggered when a mobile object collides with water.

Event Data
----------------------------------------------------------------------------------------------------

inWater
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

`boolean`_. Read-only. 

mobile
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

`tes3mobileObject`_. Read-only. Mobile object which collided.

.. _`tes3mobileObject`: ../../lua/type/tes3mobileObject.html
