This event is called when an animation is triggered using the LuaEvent key.

```lua
--- @param e animationTriggerEventData
local function animationTriggerCallback(e)
end
event.register(tes3.event.animationTrigger, animationTriggerCallback)
```

!!! tip
	This event can be filtered based on the **`name`** event data.

!!! tip
	An event can be claimed by setting `e.claim` to `true`, or by returning `false` from the callback. Claiming the event prevents any lower priority callbacks from being called.

## Event Data

* `name` (string): *Read-only*. The name of the animation triggered.
* `param` (string, nil): *Read-only*. The parameter string associated with the animation.
* `reference` ([tes3reference](../types/tes3reference.md)): *Read-only*. The reference that the animation has been triggered on.
