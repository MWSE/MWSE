This event is triggered when keyframes have been parsed, but before they have been loaded into animation groups. It can be used to add, remove, or convert file keyframe data.

```lua
--- @param e keyframesParsedEventData
local function keyframesParsedCallback(e)
end
event.register(tes3.event.keyframesParsed, keyframesParsedCallback)
```

!!! tip
	This event can be filtered based on the **`path`** event data.

!!! tip
	An event can be claimed by setting `e.claim` to `true`, or by returning `false` from the callback. Claiming the event prevents any lower priority callbacks from being called.

## Event Data

* `path` (string): The path to the keyframes file, relative to Data Files\Meshes.
* `textKeys` ({time: number, key: string, value: string}[]): An array of the loaded text key definition. Entries can be modified or inserted as tables with the keys `time`, `key`, and `value`.
