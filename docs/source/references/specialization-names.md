# Specialization Names

!!! tip
	These values are available in Lua by their index in the `tes3.specializationName` table. For example, `tes3.specializationName[0]` has a value of `"combat"`.

Index | Value
----- | -----------
-1    | `"invalid"`
0     | `"combat"`
1     | `"magic"`
2     | `"stealth"`
