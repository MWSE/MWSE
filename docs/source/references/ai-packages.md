# AI Packages

!!! tip
	These values are available in Lua by their index in the `tes3.aiPackage` table. For example, `tes3.aiPackage.none` has a value of `-1`.

Index    | Value
-------- | -----
none     | `-1`
wander   | `0`
travel   | `1`
escort   | `2`
follow   | `3`
activate | `4`
