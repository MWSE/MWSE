# AI Behavior States

!!! tip
	These values are available in Lua by their index in the `tes3.aiBehaviorState` table. For example, `tes3.aiBehaviorState.undecided` has a value of `-1`.

Index     | Value
--------- | -----
undecided | `-1`
hello     | `0`
idle      | `2`
attack    | `3`
avoid     | `4`
flee      | `6`
walk      | `8`
greet     | `12`
