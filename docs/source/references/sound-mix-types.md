# Sound Mix Types

!!! tip
	These values are available in Lua by their index in the `tes3.soundMix` table. For example, `tes3.soundMix.voice` has a value of `1`.

Index     | Value
--------- | -----
master    | `0`
voice     | `1`
effects   | `2`
footsteps | `3`
music     | `4`
