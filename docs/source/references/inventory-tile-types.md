# Inventory Tile Types

!!! tip
	These values are available in Lua by their index in the `tes3.inventoryTileType` table. For example, `tes3.inventoryTileType.magicBartered` has a value of `5`.

Index         | Value
------------- | -----
default       | `0`
equipped      | `1`
magic         | `2`
magicEquipped | `3`
bartered      | `4`
magicBartered | `5`
