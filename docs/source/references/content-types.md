# Content Types

!!! tip
	These values are available in Lua by their index in the `tes3.contentType` table. For example, `tes3.contentType.image` has a value of `"image"`.

Index  | Value
------ | ----------
image  | `"image"`
layout | `"layout"`
model  | `"model"`
rect   | `"rect"`
text   | `"text"`
