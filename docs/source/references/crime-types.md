# Crime Types

!!! tip
	These values are available in Lua by their index in the `tes3.crimeType` table. For example, `tes3.crimeType.werewolf` has a value of `7`.

Index      | Value
---------- | -----
attack     | `1`
killing    | `2`
stealing   | `3`
pickpocket | `4`
theft      | `5`
trespass   | `6`
werewolf   | `7`
