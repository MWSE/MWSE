# Spell Types

!!! tip
	These values are available in Lua by their index in the `tes3.spellType` table. For example, `tes3.spellType.disease` has a value of `3`.

Index   | Value
------- | -----
spell   | `0`
ability | `1`
blight  | `2`
disease | `3`
curse   | `4`
power   | `5`
