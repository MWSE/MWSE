# Weathers

!!! tip
	These values are available in Lua by their index in the `tes3.weather` table. For example, `tes3.weather.snow` has a value of `8`.

Index    | Value
-------- | -----
clear    | `0`
cloudy   | `1`
foggy    | `2`
overcast | `3`
rain     | `4`
thunder  | `5`
ash      | `6`
blight   | `7`
snow     | `8`
blizzard | `9`
