# Dynamic Effect Types

!!! tip
	These values are available in Lua by their index in the `tes3.dynamicEffectType` table. For example, `tes3.dynamicEffectType.ambientLight` has a value of `0`.

Index            | Value
---------------- | -----
ambientLight     | `0`
directionalLight | `1`
pointLight       | `2`
spotLight        | `3`
textureEffect    | `4`
