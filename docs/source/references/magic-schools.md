# Magic Schools

!!! tip
	These values are available in Lua by their index in the `tes3.magicSchool` table. For example, `tes3.magicSchool.destruction` has a value of `2`.

Index       | Value
----------- | -----
alteration  | `0`
conjuration | `1`
destruction | `2`
illusion    | `3`
mysticism   | `4`
restoration | `5`
none        | `6`
