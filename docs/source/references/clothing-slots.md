# Clothing Slots

!!! tip
	These values are available in Lua by their index in the `tes3.clothingSlot` table. For example, `tes3.clothingSlot.leftGlove` has a value of `6`.

Index      | Value
---------- | -----
pants      | `0`
shoes      | `1`
shirt      | `2`
belt       | `3`
robe       | `4`
rightGlove | `5`
leftGlove  | `6`
skirt      | `7`
ring       | `8`
amulet     | `9`
