# Attachment Types

!!! tip
	These values are available in Lua by their index in the `tes3.attachmentType` table. For example, `tes3.attachmentType.activator` has a value of `9`.

Index     | Value
--------- | -----
lock      | `3`
variable  | `6`
actor     | `8`
activator | `9`
