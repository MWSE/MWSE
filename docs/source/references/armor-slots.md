# Armor Slots

!!! tip
	These values are available in Lua by their index in the `tes3.armorSlot` table. For example, `tes3.armorSlot.boots` has a value of `5`.

Index         | Value
------------- | -----
helmet        | `0`
cuirass       | `1`
leftPauldron  | `2`
rightPauldron | `3`
greaves       | `4`
boots         | `5`
leftGauntlet  | `6`
rightGauntlet | `7`
shield        | `8`
leftBracer    | `9`
rightBracer   | `10`
