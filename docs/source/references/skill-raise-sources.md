# Skill Raise Sources

!!! tip
	These values are available in Lua by their index in the `tes3.skillRaiseSource` table. For example, `tes3.skillRaiseSource.progress` has a value of `"progress"`.

Index    | Value
-------- | ------------
book     | `"book"`
progress | `"progress"`
training | `"training"`
