# Effect Event Types

!!! tip
	These values are available in Lua by their index in the `tes3.effectEventType` table. For example, `tes3.effectEventType.bool` has a value of `0`.

Index        | Value
------------ | -----
bool         | `0`
boolean      | `0`
integer      | `1`
int          | `1`
float        | `2`
modStatistic | `3`
