# Apparatus Types

!!! tip
	These values are available in Lua by their index in the `tes3.apparatusType` table. For example, `tes3.apparatusType.calcinator` has a value of `2`.

Index           | Value
--------------- | -----
mortarAndPestle | `0`
alembic         | `1`
calcinator      | `2`
retort          | `3`
