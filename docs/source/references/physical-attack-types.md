# Physical Attack Types

!!! tip
	These values are available in Lua by their index in the `tes3.physicalAttackType` table. For example, `tes3.physicalAttackType.none` has a value of `0`.

Index      | Value
---------- | -----
none       | `0`
slash      | `1`
chop       | `2`
thrust     | `3`
projectile | `4`
creature1  | `5`
creature2  | `6`
creature3  | `7`
