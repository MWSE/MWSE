# Dialogue Types

!!! tip
	These values are available in Lua by their index in the `tes3.dialogueType` table. For example, `tes3.dialogueType.voice` has a value of `1`.

Index    | Value
-------- | -----
topic    | `0`
voice    | `1`
greeting | `2`
service  | `3`
journal  | `4`
