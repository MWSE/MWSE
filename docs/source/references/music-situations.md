# Music Situations

!!! tip
	These values are available in Lua by their index in the `tes3.musicSituation` table. For example, `tes3.musicSituation.explore` has a value of `0`.

Index           | Value
--------------- | -----
explore         | `0`
combat          | `1`
uninterruptible | `2`
