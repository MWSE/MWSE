# Animation Start Flags

!!! tip
	These values are available in Lua by their index in the `tes3.animationStartFlag` table. For example, `tes3.animationStartFlag.normal` has a value of `0`.

Index         | Value
------------- | -----
normal        | `0`
immediate     | `1`
immediateLoop | `2`
