# Skill Types

!!! tip
	These values are available in Lua by their index in the `tes3.skillType` table. For example, `tes3.skillType.miscellaneous` has a value of `2`.

Index         | Value
------------- | -----
major         | `0`
minor         | `1`
misc          | `2`
miscellaneous | `2`
