# Damage Sources

!!! tip
	These values are available in Lua by their index in the `tes3.damageSource` table. For example, `tes3.damageSource.magic` has a value of `"magic"`.

Index       | Value
----------- | ---------------
attack      | `"attack"`
fall        | `"fall"`
magic       | `"magic"`
script      | `"script"`
shield      | `"shield"`
suffocation | `"suffocation"`
