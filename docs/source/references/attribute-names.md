# Attribute Names

!!! tip
	These values are available in Lua by their index in the `tes3.attributeName` table. For example, `tes3.attributeName[0]` has a value of `"strength"`.

Index | Value
----- | ----------------
3     | `"agility"`
5     | `"endurance"`
1     | `"intelligence"`
7     | `"luck"`
6     | `"personality"`
4     | `"speed"`
0     | `"strength"`
2     | `"willpower"`
